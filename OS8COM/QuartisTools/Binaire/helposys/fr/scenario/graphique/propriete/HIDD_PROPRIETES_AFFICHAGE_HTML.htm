Propriétés de l'action de type Affichage HTML

 

Une action Affichage HTML  est définie par :

 

Nom

Nom de l'action Affichage HTML.

Description

Description de l'action Affichage HTML.

Type d'afficheur

L'afficheur a 16 ou 40 colonnes.

Si gestion des deux  types d'afficheurs, le message peut être différent suivant les deux types d'afficheurs.

Fichier HTML

Nom de la page Html.

Type du canal pour les requêtes

Indiquer le type du canal (canal lent ou canal rapide) pour afficher le résultat de l'exécution d'une requête.

Définition Xml

Liste des balises présentes dans le fichier HTML.

Possibilité d'accrocher une mémoire ou une valeur ou une requête suivant le type de la balise.
