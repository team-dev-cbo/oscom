Editeur LPX

 

L'éditeur LPX permet d'écrire un programme ou un module en langage LPX.

Le programme LPX est une suite d'instructions et est appelé dans une action.

Le module LPX est une bibliothèque de sous programmes, de types, de constantes et est utilisé dans un programme LPX.

 

 

Structure générale d'un programme LPX :

 

WITH Liste des modules;

 

PROCEDURE Nom de Procédure ;

 

TYPE

 Déclaration(s) de type ;

 

CONST

 Déclaration(s) de constante ;

 

VAR

 Déclaration(s) de variable ;

 

BEGIN

 Instructions ;

 

END Nom de Procédure ;

 

Structure générale d'un module LPX :

 

WITH Liste des modules;

 

MODULE Nom du module ;

 

EXPORT liste ( procedure, fonction, type, variable, constante) ;

 

TYPE

 Déclaration(s) de type ;

 

CONST

 Déclaration(s) de constante ;

 

VAR

 Déclaration(s) de variable ;

 

FUNCTION

 Déclaration(s) des fonctions ;

 

PROCEDURE

 Déclaration(s) des procédures ;

 

END Nom du module ;

 

 

Eléments de syntaxe :

Toutes les instructions sont suivies d'un point virgule, à l'exception de certains mots réservés.

Pour ajouter une ligne de commentaire , il suffit de précéder le texte par //

 

Insertion :

Il est possible d'insérer automatiquement le prototype d'une fonction LPX , le nom d'une mémoire et le nom d'un Lien d'Osys Bornéo.

 

Compilation :

Pour lancer la compilation , il suffit d'appuyer sur le bouton .

Si des erreurs sont trouvées lors de la compilation , celles-ci sont répertoriées .Un double click sur une erreur permet de localiser l'erreur dans l'édition. Si une erreur est sélectionnée , l'appui sur la touche F1 donne une indication sur la source d'erreur rencontrée.
