HtmlImageLien, Fonction

 

Insère une image qui renvoie à une page Html lorsque l'on clique sur celle-ci.

 

 

Syntaxe

 

HtmlImageLien( p_image,p_lien : GrandeChaine ; p_style : Integer )

 

Elément

Description

 

p_image

Fichier image

p_lien

Fichier de la page Html.

p_style

Position du texte par rapport à l'image : HAUT,BAS,MILIEU.

 

Remarque

Les fichiers image et lien doivent être présents dans le répertoire Web. S'ils se trouvent dans un sous répertoire, préciser le chemin complet. ( ex: http://.... )

 

 

 

Exemple

 

WITH  Html ;

 

PROCEDURE Exemple ;

 

VAR 

 i : Integer ;

 

BEGIN

 

 // début de la page HTML

 HtmlPageDebut( "Exemple","",0 );

 

 // titre de la page

 HtmlTitre( "Exemple", 1,GRAS+ROUGE+ITALIQUE+CENTRE ) ;

 

 // ligne suivante

 HtmlLigneFin;

 

 // texte

 HtmlTexte( "site osys : " , ROSE );

 

 // image lien

 HtmlImageLien( "term.gif","default.htm" , MILIEU ); 

 

 // fin de la page HTML

 HtmlPageFin;

 

END Exemple ;
