DeFormaterTemps, Fonction

 

Conversion d’une date+heure (stockée dans une chaîne formatée) en entier. La valeur renvoyée correspond au nombre de secondes écoulées depuis le 1er janvier 1970. Renvoie (-1) si la fonction détecte une erreur de conversion.

 

 

Syntaxe

 

DeFormaterTemps ( Format, Date : GrandeChaine ) return INTEGER

 

Elément

Description

 

Format

Format de présentation de la date

Date

Date à convertir

 

Format

Description

 

%a

Retourne le nom en abrégé du jour de la semaine

%A

Retourne le nom en clair du jour de la semaine

%b

Retourne le nom du mois en abrégé

%B

Retourne le nom en clair du mois

%d

Retourne le n° du jour (01-31)

%H

Retourne l’heure sur 24 h (00-23)

%I

Retourne l’heure sur 12 h (00-11)

%j

Retourne le n° du jour de l'année (001-366)

%m

Retourne le n° du mois (01-12)

%M

Retourne les minutes (00-59)

%S

Retourne les secondes (00-59)

%y

Retourne l’année sur 2 chiffres (00-99)

%Y

Retourne l’année sur 4 chiffres (ex: 1993)

%D

Retourne la date( jj/mm/aaaa )

%T

Retourne l’heure ( HH:MM:SS )

 

Remarque

Un test sur la validité de la date/heure est réalisé si le format est « %D %T ».

 

Exemple

 

WITH stdlib;

VAR Temps : INTEGER;

...

Temps := DeFormaterTemps ( "%d-%m-%Y", "29-09-1999" );
