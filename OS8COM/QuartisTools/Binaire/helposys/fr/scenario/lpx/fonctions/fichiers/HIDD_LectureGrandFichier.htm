LectureGrandFichier, Fonction

Lecture d’une chaîne de caractères (de 2047 caractères maximum) à partir d'un fichier.

La fonction lit la chaîne de caractères à la position du pointeur courant du fichier.

La chaîne est lue jusqu'au prochain LF, puis le pointeur de fichier se déplace pour lire la prochaine ligne.

Si la fin de fichier est détectée, la fonction renverra EOF.

Syntaxe

LectureGrandFichier (Fichier : T_Fichier) return Chaine2048

 

Elément

Description

 

Fichier

Fichier est le pointeur du fichier.

 

Exemple

With stdlib ;

Procedure ExempleFic

Var

Fichier : T_Fichier;

ChaineLue : Chaine2048;

Begin

Fichier := OuvreFichierLecture ("C:\Import\Article.txt");

ChaineLue := LectureGrandFichier (Fichier);

While (ChaineLue<>EOF) do

         EcritMessageErreur("ChaineLue:"+ChaineLue );

ChaineLue := LectureGrandFichier (Fichier);

End While;

FermeFichier(Fichier);

 

End ExempleFic ;
