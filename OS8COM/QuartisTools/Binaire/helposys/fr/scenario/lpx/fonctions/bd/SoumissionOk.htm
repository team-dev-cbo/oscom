SoumissionOk, Fonction

Test si la requête a bien été exécutée par la base de données  (même si erreur d'exécution) ou non.

Par exemple, la requête n'a pas pu être exécuté à cause d'une coupure réseau entre la base de données et le moteur Quiartis Tools.

Retourne TRUE si la requête a bien été exécutée par la base de données sinon retourne FALSE.

Syntaxe

SoumissionOk( ReponseSQL : T_ReponseSQL ) return BOOLEAN

 

Elément

Description

 

ReponseSQL

Objet réponse SQL à tester.

 

Exemple

With  database ;

Procedure Exemple;

Var

    ReponseSQL : T_ReponseSQL;

    Fermeture : Boolean;

 

Begin

    ReponseSQL :=ExecuteRequeteSQL("SELECT * FROM Personnel WHERE Matricule ='150'");

    if ( SoumissionOk ( ReponseSQL ) ) THEN

.....Traitement 1

   else

.....Traitement 2

   end if ;

   Fermeture := FermeReponseSQL( ReponseSQL ) ;

End Exemple;
