CGILitValeur, Fonction

 

Lecture du contenu d’un élément d’un formulaire dans une page Html.

 

 

Syntaxe

 

CGILitValeur( p_nom : GrandeChaine ) RETURN GrandeChaine

 

Elément

Description

 

p_nom

Nom de l’élément du formulaire

 

 

Remarque

Le contenu renvoyé par CGILitValeur n’excède pas 255 caractères. Pour une lecture d’une taille plus importante, utiliser la fonction CGILitValeurEx.

Si un élément comporte plusieurs valeurs ( Liste multi-sélection ) , les valeurs sont séparées par une virgule.

 

Exemple

 

WITH  Html ;

 

PROCEDURE Exemple ;

 

VAR

 

l_chaine : GrandeChaine ;

 

 

BEGIN

 

 // début de la page

 HtmlPageDebut( "Essai","",0 );

 

 // début du formulaire

 HtmlFormDebut( "SFAO") ;

 

 // insertion d'une liste

HtmlListeDebut("Liste" , 3 , TRUE ) ; // liste multi-sélection

 

// Ajout dans la liste

HtmlListeAjout("Zidane" ,"10", FALSE ) ;

HtmlListeAjout("Petit" ,"8", FALSE ) ;

HtmlListeAjout("Dugarry" ,"9", FALSE ) ;

HtmlListeAjout("Thuram" , "2",FALSE ) ;

HtmlListeAjout("Barthez" , "1",FALSE ) ;

 

// fin de la liste

HtmlListeFin ;

 

// insertion d'un bouton de validation pour l'envoi vers Osys Bornéo

HtmlBoutonValide( "bouton","Validation" );

 

// fin du formulaire

HtmlFormFin ;

 

// fin de la page

HtmlPageFin ;

 

AttenteSaisie ; // attente de l'appui sur le bouton validation

 

// lecture des éléments de la liste sélectionnés

l_chaine := CGILitValeur( "Liste" );

& l_chaine ;

 

// page de réponse

 // début de la page HTML

 HtmlPageDebut( "Reponse" ,"",0 );

 

 // réponse

 HtmlTexteASCII( "Valeur de l’élément sélectionné : " + l_chaine , ROUGE );

 

 // fin de la page HTML

 HtmlPageFin;

 

 

END Exemple ;
