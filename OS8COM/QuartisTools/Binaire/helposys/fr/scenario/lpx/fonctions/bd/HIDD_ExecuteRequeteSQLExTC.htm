ExecuteRequeteSQLExTC, Fonction

 

Exécute une requête pour un type de canal donné (Le type du canal est lent ou rapide).

Renvoie les renseignements sur l’exécution de la requête, 0 si problème de connexion ADO.

 

 

Syntaxe

 

ExecuteRequeteSQLExTC( ChaineSQL : ChaineEx, TypeCanal : Integer ) return T_ReponseSQL.

 

Elément

Description

 

ChaineSQL 

Requête SQL.

TypeCanal 

Type du canal (LENT ou RAPIDE).

With database ;

Var Reponse : T_ReponseSQL ;
