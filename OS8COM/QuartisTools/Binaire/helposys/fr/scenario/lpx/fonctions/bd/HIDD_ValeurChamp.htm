ValeurChamp, Fonction

 

Donne le contenu d’un champ d’un enregistrement spécifié.

 

 

Syntaxe

 

ValeurChamp( ReponseSQL : T_ReponseSQL ; Ligne, Colonne : INTEGER ) return GrandeChaine

 

Elément

Description

 

ReponseSQL 

Resultat de la requête

Ligne

N° d’enregistrement (supérieur ou égale à 1)

Colonne 

N° de champ (supérieur ou égale à 1)

 

 

Exemple

 

With database ;

Procedure Exemple;

Var

       Resultat:GrandeChaine;

       ReponseSQL : T_ReponseSQL;

       Fermeture : Boolean;

 

Begin

   ReponseSQL :=ExecuteRequeteSQL("SELECT * FROM Personnel WHERE Matricule='150'");

   if ( ExecuteOk( reponseSQL ) ) THEN

          Resultat := ValeurChamp( ReponseSQL,1,1) ;

    end if ;

    Fermeture := FermeReponseSQL( ReponseSQL ) ;

End Exemple;
