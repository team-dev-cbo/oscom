ReserveCanal, Fonction

 

Méthode obsolète à partir de la version V3.0.0.0 de Bornéo.

 

Réserve un canal rapide pour l’accès à la base de données.

Rq: Le canal reste réservé tant que  LibereCanal n’a pu être exécutée

 

Renvoie le numéro du canal si l’accès à la base de données a été réservé sinon renvoie 0.

 

 

Syntaxe

 

ReserveCanal return T_CanalBD;

 

Exemple

 

With   database ;

Var NumCanal : T_CanalBD ;

 

 NumCana1:=ReserveCanal;
