FermeFenetreUtilisateur, Procédure

Fermeture de la fenêtre 'NomFenetre' pour l'utilisateur connecté 'NomUtilisateur'.

Syntaxe

FermeFenetreUtilisateur (NomFenetre : GrandeChaine, NomUtilisateur : GrandeChaine  )

 

Elément

Description

 

NomFenetre

Nom de la fenêtre qui doit être fermée.

NomUtilisateur

Nom de l'utilisateur

 

 

Exemple

 

With stdlib;

 

 // Fermeture de la fenêtre pour l'utilisateur 'NomUtilisateur'

 FermeFenetreUtilisateur("NomFenetre",  "NomUtilisateur" );
