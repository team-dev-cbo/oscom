MemoireGroupeExiste, Fonction

Test si la mémoire "NomMemoire" du groupe "NomGroupeMemoire" existe.

Renvoie TRUE si la mémoire existe, FALSE sinon.

Syntaxe

MemoireGroupeExiste(NomGroupeMemoire : GrandeChaine, NomMemoire : GrandeChaine) return Boolean;

Elément

Description

 

NomGroupeMemoire

Nom du groupe de la mémoire

NomMemoire

Nom de la mémoire

Exemple

WITH stdlib;

PROCEDURE  Exemple;

VAR

res : Boolean;

BEGIN

res :=  MemoireGroupeExiste( "GroupeMemoire1","memoire1" ) ;

END Exemple;
