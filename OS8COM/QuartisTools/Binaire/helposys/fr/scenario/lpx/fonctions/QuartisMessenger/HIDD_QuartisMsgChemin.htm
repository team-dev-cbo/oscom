QuartisMsgChemin, Fonction

Renseigne dans la mémoire système "Chemin QuartisMsg" le chemin de l'application Quartis Messenger Client.

Syntaxe

QuartisMsgChemin( CheminQuartisMsg : GrandeChaine ) return Integer

Retour

 1 :  OK
-1 :  Erreur chemin
-2 :  Erreur répertoire base de données
-3 :  Erreur base de données

 

Elément

Description

CheminQuartisMsg

Chemin de l'application

 

Exemple

With stdlib, stdstring ;

Procedure Quartis ;

Var

   // variables

   l_res : Integer ;

   i : Integer ;

   l_res1 : Boolean ;

   l_chaine : GrandeChaine ;

 

Begin

 

   // Initialisation du chemin de Quartis Messenger Client

   l_chaine  := LitMemoire( "Chemin QuartisMsg" ) ;

 

   If  l_chaine <> "D:\QuartisMsg\Binaire"  Then

        l_res := QuartisMsgChemin( "D:\QuartisMsg\Binaire" ) ;

   End If ;

 

   // Ajout de Messages à envoyer à Quartis Messenger Serveur

   For i:=1 To 10 Do

        // Durée des Messages : 3 minutes ( 180 = 3x60 )

        l_res1  := QuartisMsgEnvoieMessage( "Liste1" , "C" + "Message"+ Str( i ) + "_Liste1" , TempsSysteme + 180 ) ;

   End For ;

 

End  Quartis ;
