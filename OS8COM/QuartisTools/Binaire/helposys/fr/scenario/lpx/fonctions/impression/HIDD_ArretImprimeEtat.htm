ArretImprimeEtat, Fonction

Demande d'arrêter l'impression de l'état au driver état.

Remarque :

Avant de demander l'arrêt d'une impression d'un état il faut envoyer un "ImprimeEtat".

Syntaxe

ArretImprimeEtat ( NomDriver  :GrandeChaine ) return Integer  

Retour :     

0  si la demande a été envoyée au driver état

1  si le driver état  n'existe pas

 

Elément

Description

 

NomDriver

Nom du driver état

 

Exemple

With stdlib ;

Procedure TestImprimeEtat ;

Var

    l_retour: Integer;

Begin  

      l_retour:=ImprimeEtat("Etat1","PDF","C:\temp\info.pdf","DrvEtat1");

      .................

      l_retour:=ArretImprimeEtat("DrvEtat1");

 End TestImprimeEtat ;
