ExecScenario, Procédure

 

Exécute un nouveau scénario LPX.

Pour la liste des paramètres :

- Le séparateur entre deux paramètres est l'espace

- Si le paramètre contient des espaces, il faut ajouter des guillemets

Exemple :

ListeParam = ValParam1 ValParam2 "Valeur du parametre 3" ValParam4

Param1 = ValParam1

Param2 = ValParam2

Param3 = Valeur du parametre 3

Param4 = ValParam4

 

Syntaxe

 

ExecScenario(NomScénario : GrandeChaine, ParamètreScénario : GrandeChaine)

 

Elément

Description

 

NomScénario

Nom du scénario à exécuter.

ParamètreScénario

Liste des Paramètres du scénario.

 

 

Exemple

 

WITH stdlib ;

PROCEDURE Exemple1;

VAR

ListeParam : GrandeChaine;

BEGIN

ListeParam:="ValParam1 ValParam2 "+GU+"Val Param 3 "+GU+" Param4";

ExecScenario("Exemple2", ListeParam );

 

END Exemple1;

 

 

WITH stdlib ;

PROCEDURE Exemple2;

VAR

Param1, Param2, Param3,Param4 : GrandeChaine;

BEGIN

Param1:=LitParam(1);

Param2:=LitParam(2);

Param3:=LitParam(3);

Param4:=LitParam(4);

EcritMessageErreur("Param1="+Param1+" Param2="+Param2+" Param3="+Param3+" Param4="+Param4);

 

END Exemple2;

 

 

Le résultat de l'exécution du scenario LPX "Exemple1" est :

Param1=ValParm1 Param2=ValParam2 Param3=Val Param 3 Param4=Param4
