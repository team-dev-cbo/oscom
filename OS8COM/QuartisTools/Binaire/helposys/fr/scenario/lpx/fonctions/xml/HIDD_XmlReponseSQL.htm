XmlReponseSQL, Fonction

 

Ecrit le résultat d’une requête dans une structure xml.

 

 

Syntaxe

 

XmlReponseSQL( p_xml : T_Xml ; p_repSQL : T_ReponseSQL ; p_prototype : GrandeChaine )

RETURN Integer

 

Elément

Description

 

p_xml

identifiant structure xml

p_repSQL

Identifiant résultat d’une requête

p_prototype

Prototype de la structure xml ( voir remarque )

 

 

Valeur retournée

0 : OK ,

>0 : nombre de colonnes de la réponse SQLnon utilisées,

-1 : erreur identifiant structure xml,

-2 : erreur identifiant réponse SQL,

-3 : erreur prototype.

 

Remarque

Syntaxe du prototype :

BaliseElement ; BaliseEnregistrement ; BaliseChamp1 ; BaliseChamp2 ;… ; BaliseChampN

 

Avec BaliseElement : Nom de la balise délimitant l’élément dans la structure Xml.

 BaliseEnregistrement : Nom de la balise délimitant une ligne de la réponse SQL.

 BaliseChamp1 : Nom de la balise délimitant la 1ère colonne d’une ligne de la réponse SQL.

 BaliseChamp2 : Nom de la balise délimitant la 2ième colonne d’une ligne de la réponse SQL.

 BaliseChampN : Nom de la balise délimitant la Nième colonne d’une ligne de la réponse SQL.

 

Dans la structure xml , BaliseEnregistrement est un fils de BaliseElement .

BaliseChamp1 , BaliseChamp2  et BaliseChampN sont des fils de BaliseEnregistrement .

 

 

Exemple

 

With XML,HTML,OSYS_Utilitaires,stdlib,OSYS_Base_Donnees;

 

Procedure MF_Web_ShowTrace;

Var

Req : TresGrandeChaine;

Rep : T_ReponseSQL;

Res : Boolean;

 

DateActuelle : Integer;

 

XMLListe : T_Xml;

Resultat : Integer;

Begin

//----- Récupération de la date actuelle

DateActuelle := TempsSysteme;

 

//----- Sélection des évènements de pointage (on récupère la couleur rouge pour les anomalies)

Req := "SELECT Format (DATE_POINTAGE,'hh:mm:ss'),MATRICULE,SAISIE,SCENARIO,DESIGNATION," +

" IIf(FLAG_ANOMALIE='O','"+ROUGE_+"','"+NOIR_+"')" +

" FROM T_EVT_POINTAGE" +

" WHERE DATE_POINTAGE >= " + FormaterTempsBD (

DeformaterTemps ("%Y/%m/%d",

FormaterTemps ("%Y/%m/%d",DateActuelle)

)

) +

" ORDER BY DATE_POINTAGE DESC";

 

Rep := ExecuteRequeteSQL (Req);

 

//----- Création de la structure XML

XMLListe := XmlCree("");

 

//----- On ajoute l'élément XML

Resultat := XmlElementAjoute (XMLListe,"XML","");

 

 

//----- On écrit le résultat de la requete dans la liste XML

Resultat := XmlReponseSQL( XMLListe, Rep, "T_EVT_POINTAGE;"+"LIGNE;"+"DATE_POINTAGE;MATRICULE;SAISIE;SCENARIO;DESIGNATION;COULEUR");

 

//----- On "mixe" la page HTML avec la liste XML

Resultat := HtmlXml(LitMemoire("Répertoire application") +

ConstPathSupportWeb + "trace.htm", XMLListe);

 

// Resultat := XmlFichierEcrit (XMLListe,"d:\trace.xml");

 

//----- Fermeture de la liste XML

Resultat := XmlFerme (XMLListe);

 

//----- Fermeture de la réponse SQL

Res := FermeReponseSQL (Rep);

End MF_Web_ShowTrace;
