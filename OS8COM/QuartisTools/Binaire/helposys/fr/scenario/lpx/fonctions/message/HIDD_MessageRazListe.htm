MessageRazListe, Fonction

 

Remise à zéro de la liste des messages.

 

 

Syntaxe

 

MessageRazListe

 

Remarque

Osys Bornéo initialise la liste des messages avec un fichier du répertoire Mess de l’application.

Le nom du fichier doit contenir la langue courante et l’extension mes. ( ex : nomFic_fr.mes pour la langue française ).

A chaque appel de la fonction, une actualisation du fichier est effectuée.

 

Exemple

 

With  Mess ;

 

Procedure test1 ;

 

Var

 l_message : GrandeChaine ;

 l_res : Integer ;

 

Begin

 

 // remise à zéro de la liste

 MessageRazListe ;

 

 // ajout de messages

 l_res := MessageAjoute( "M1" , "Message1" );

 l_res := MessageAjoute( "M2" , "Message2" );

 l_res := MessageAjoute( "M3" , "Message3" );

 l_res := MessageAjoute( "M4" , "Message5" );

 

 // lecture des messages

 l_res := Message( "M1" , @ l_message );

 l_res := Message( "M2" , @ l_message );

 l_res := Message( "M3" , @ l_message );

 l_res := Message( "M4" , @ l_message );

 

 // modification d'un message

 l_res := MessageModifie( "M1" , "Message1 modifié" );

 // lecture du message

 l_res := Message( "M1" , @ l_message );

 

 // suppression d'un message

 l_res := MessageSupprime( "M2" );

 // lecture du message

 l_res := Message( "M2" , @ l_message );

 

 

End test1 ;
