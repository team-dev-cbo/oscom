MemoireReponseSQL, Fonction

Transfert le contenu d'un T_ReponseSQL dans une mémoire. Avant le transfert, le contenu de la  mémoire est au préalable détruit.

Retourne TRUE si OK, FALSE sinon ( si la mémoire n'existe pas ou si le nombre maximum de valeurs de la mémoire est inférieur au nombre total d'enregistrements de la requête).

Syntaxe

MemoireReponseSQL(NomMemoire : GrandeChaine; ReponseSQL : T_ReponseSQL) return Boolean;

 

Elément

Description

 

NomMemoire

Nom de la Mémoire

ReponseSQL

Réponse de l'exécution d'une requête

    Fermeture : Boolean;

"Traitement"

    Fermeture := FermeReponseSQL( Reponse ) ;

 

End  Exemple ;
