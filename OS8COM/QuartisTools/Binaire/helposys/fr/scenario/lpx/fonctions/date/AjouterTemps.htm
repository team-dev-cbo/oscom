AjouterTemps, Fonction

Ajoute la durée à la date passée en paramètre ou la retranche si la durée est positive ou négative.

Si la fonction détecte une erreur de conversion, la date renvoyée est vide. Sinon, renvoie la date calculée au format passée en paramètre.

Syntaxe

AjouterTemps ( Format : GrandeChaine ,  Date: GrandeChaine, Duree : Integer ) return GrandeChaine

 

Elément

Description

 

Format

Format de présentation de la date

Date

Date à convertir

Durée

Durée en seconde

 

Format

Description

 

%a

Retourne le nom en abrégé du jour de la semaine

%A

Retourne le nom en clair du jour de la semaine

%b

Retourne le nom du mois en abrégé

%B

Retourne le nom en clair du mois

%d

Retourne le n° du jour (01-31)

%H

Retourne l’heure sur 24 h (00-23)

%I

Retourne l’heure sur 12 h (00-11)

%j

Retourne le n° du jour de l'année (001-366)

%m

Retourne le n° du mois (01-12)

%M

Retourne les minutes (00-59)

%S

Retourne les secondes (00-59)

%y

Retourne l’année sur 2 chiffres (00-99)

%Y

Retourne l’année sur 4 chiffres (ex: 1993)

%D

Retourne la date( jj/mm/aaaa )

%T

Retourne l’heure ( HH:MM:SS )

 

Valeur retournée

La date au format passée en paramètre.

Remarque

Un test sur la validité de la date/heure est réalisé si le format est « %D %T ».

La date calculée n'est pas convertie automatiquement lors du passage de l'heure d'hiver à l'heure d'été et vis versa.

Exemple :

Format = "%Y/%m/%d H:M:S"

Date="2007/03/25 01:00:00" (Jour de passage de l'heure d'hiver à l'heure d'été à 2h )

Duree = 3600 (= 1 heure)

Date retournée = "2007/03/25 02:00:00"

Exemple

 

WITH stdlib;

VAR Date : GrandeChaine;

...

// Ajouter 1 heure

Date :=AjouterTemps ( "%d/%m/%Y H:M:S", "31/03/2007 03:00:00", 3600 );

// supprimer 1 heure

Date :=AjouterTemps ( "%d/%m/%Y H:M:S", "31/03/2007 03:00:00", - 3600 );
