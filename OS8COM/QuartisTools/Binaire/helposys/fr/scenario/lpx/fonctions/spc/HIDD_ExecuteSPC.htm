ExecuteSPC, Procédure

Effectue une mesure sur une production donnée avec SPC Vision.

Syntaxe

ExecuteSPC ( p_param : T_ParamSPC ) ;

 

Elément

Description

 

p_param

Structure définissant les paramètres de la mesure à effectuer.

 

Exemple

WITH stdlib;

Procedure TestSPC;

Var

     param : T_ParamSPC ;

     chaine : GrandeChaine;

Begin

param.langage := "FR";

param.flagAutoClose := False;

param.codeAtelier := "ProdStd" ;

param.codeMachine := "Fraiseuse";

param.codeProduit := "Ecrous 12" ;

param.codeGammeControle := "MO Ecrous 5" ;

param.codeOF := "" ;

param.codeOperateur := "003" ;     

ExecuteSPC ( param ) ;

End TestSPC;
