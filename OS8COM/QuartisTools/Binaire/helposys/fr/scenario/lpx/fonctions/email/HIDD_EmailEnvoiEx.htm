EmailEnvoiEx, Fonction

 

Envoi d'un courrier électronique accompagné d'un fichier.

 

 

Syntaxe

 

EmailEnvoiEx( p_serveur,p_destinataire,p_emetteur,p_sujet,p_fichier : GrandeChaine ;

p_texte : TresGrandeChaine )

 

Elément

Description

 

p_serveur

Nom du serveur ( Nom de la machine sur le réseau )

p_destinataire

Adresse du destinataire

p_emetteur

Nom de l'émetteur

p_sujet

Objet du courrier

p_fichier

Fichier à envoyer ( chemin complet )

p_texte

Contenu du courrier

Exemple

WITH Email ;

 

PROCEDURE Exemple ;

  

 

BEGIN

 // envoi d'un courrier avec fichier accompagné

 EmailEnvoiEx( "tintin","????@osys.fr","Le chef de service","A cheval","c:\tmp\martinet.jpg" ,

"Fini les discours, au boulot !!!" );

  

END Exemple ;
