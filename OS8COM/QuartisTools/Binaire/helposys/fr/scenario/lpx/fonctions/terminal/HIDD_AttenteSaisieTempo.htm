AttenteSaisieTempo, Fonction

Attend la réception d'un message en provenance du terminal courant.
Si au bout de X seconde(s) aucun message n'a été reçu :
                -  Le scénario continu à s'exécuter
                -  Le résultat est sauvegardé dans un T_AttenteTempo
                -  Le message reçu est vide ( récupéré par la méthode LitMessage )
 

Syntaxe

AttenteSaisieTempo (dureee: integer) return T_AttenteTempo

 

Elément

Description

Duree

Temps maximum d'attente d'un message (en secondes)

 

 

Exemple

WITH stdlib;
Procedure TestAttenteSaisieTempo;
Var
l_tempo : T_AttenteTempo;
Begin
  l_tempo:=AttenteSaisieTempo(5);
  If (LitValAttenteSaisieTempo(l_tempo) = 1) Then
       EcritTerminal("Vous avez tap : " + LitMessage);
  Else
       EcritTerminal("Erreur");
  EndIf;
End TestAttenteSaisieTempo;
