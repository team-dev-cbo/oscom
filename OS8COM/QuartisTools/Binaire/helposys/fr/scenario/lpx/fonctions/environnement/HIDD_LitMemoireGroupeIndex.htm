LitMemoireGroupeIndex, Fonction

Lecture du contenu de la mémoire "NomMemoire"  dans l'index numéro "Index"  de la liste des mémoires de la mémoire "NomMemoire".

La mémoire doit être définie dans le groupe "NomGroupeMemoire ".

Renvoie le contenu de la mémoire

ou chaîne vide si la mémoire n’existe pas dans le groupe ou si le groupe n'existe pas

ou FALSE si l'index n'existe pas dans la mémoire.

 

Syntaxe

LitMemoireGroupeIndex ( NomGroupeMemoire :GrandeChaine,  NomMemoire : GrandeChaine, Index : Integer ) return TresGrandeChaine;

 

Elément

Description

 

NomGroupeMemoire

Nom du groupe de la mémoire

NomMemoire

Nom de la mémoire

Index

Numéro de l'index qui va contenir le contenu de la mémoire

 

Exemple

WITH stdlib;

& LitMemoireGroupeIndex("Système","Date/heure action", 2);
