FermeReponseFichier, Procédure

Suppression du résultat du traitement sur le fichier (copier, supprimer, déplacer un fichier ou afficher la liste des fichiers).

 

Remarque :

Si le résultat n'est pas supprimé, il reste en mémoire.

 

Syntaxe

FermeReponseFichier (ReponseFic : T_Fichier)

 

Elément

Description

 

ReponseFic

Résultat du traitement sur le fichier.

 

 

 

Exemple

 

With stdlib ;

Procedure ExempleFic ;

Var 

l_listeFichier : ChaineEx ;

l_reponseFic : T_Fichier;

Begin

// Liste des fichiers

l_reponseFic  := ListeFichier( "c:\tmp", "*.*", 10 );

 

l_listeFichier:=ReponseListeFichier(l_reponseFic,1);

EcritMessageErreur ("Liste des fichiers ="+l_listeFichier );

 

// Liste des fichiers dont le répertoire se trouve  sur un réseau distant

l_reponseFic := ListeFichier( "\\NomServeur\Repertoire1\Repertoire_2", "*.*", 10 );

 

l_listeFichier:=ReponseListeFichier(Reponse,0);

FermeReponseFichier(l_reponse);

 

EcritMessageErreur ("Liste des fichiers ="+l_listeFichier );

End ExempleFic ;
