HtmlRadio, Fonction

 

Insère un bouton radio dans un formulaire d'une page Html.

 

 

Syntaxe

 

HtmlRadio( p_nom , p_valeur : GrandeChaine ; p_etat : BOOLEAN )

 

 

Elément

Description

 

p_nom

Nom du bouton radio

p_valeur

Valeur du bouton radio

p_etat

Etat initial du bouton radio

 

 

Remarque:

Pour définir un groupe de boutons radio avec sélection unique, il est nécessaire de lui attribuer le même nom et des valeurs différentes.

La lecture du bouton Radio sélectionné se fait par la fonction CGILitValeur .( voir exemple )

 

 

Exemple

 

WITH Html ;

 

PROCEDURE Exemple ;

 

VAR

 l_chaine : GrandeChaine ;

 

BEGIN

 

 // début de la page HTML

 HtmlPageDebut( "Essai" ,"",0 );

 

 // début du formulaire

 HtmlFormDebut("SFAO") ;

 

// insertion d'un ensemble de 4 boutons radio

 HtmlRadio( "Radio1" ,"1", TRUE );

HtmlRadio( "Radio1" ,"2", FALSE );

HtmlRadio( "Radio1" ,"3", FALSE );

HtmlRadio( "Radio1" ,"4", FALSE );

 

 

// insertion d'un bouton de validation pour l'envoi vers Osys Bornéo

HtmlBoutonValide( "bouton","Validation" );

 

// fin du formulaire

 HtmlFormFin ;

 

 // fin de la page HTML

 HtmlPageFin ;

 

 AttenteSaisie ; // attente de l'appui sur le bouton validation

 

 // lecture de l'état de la case à cocher

 l_chaine := CGILitValeur( "Radio1" );

& l_chaine ;

 

// page de réponse

 // début de la page HTML

 HtmlPageDebut( "Reponse" ,"",0 );

 

 // réponse

 HtmlTexteASCII( "Valeur du bouton radio : " + l_chaine , ROUGE );

 

 // fin de la page HTML

 HtmlPageFin;

 

 

END Exemple ;
