FormaterDateTZEnUTC, Fonction

Conversion d’une date+heure en UTC  (stockée dans un entier) en chaîne formatée pour un fuseau horaire.

L' UTC (0) est un fuseau horaire, sans décalage horaire par rapport à l'UTC (Temps universel coordonné).

Syntaxe

FormaterDateTZEnUTC (Format : GrandeChaine; Date : Integer; FuseauHoraire : GrandeChaine; GestionHeureEte : Integer ) return GrandeChaine

 

Elément

Description

 

Format

Format de présentation de la date

Date

Date TZ (au format entier) à convertir en date UTC (format chaîne)

FuseauHoraire

Nom du fuseau horaire

GestionHeureEte

Gestion ou non de l'heure d'été (0:non, 1:oui)

 

Format

Description

 

%a

Retourne le nom en abrégé du jour de la semaine

%A

Retourne le nom en clair du jour de la semaine

%b

Retourne le nom du mois en abrégé

%B

Retourne le nom du mois en clair

%d

Retourne le n° du jour (01-31)

%H

Retourne l'heure sur 24 h (00-23)

%I

Retourne l'heure sur 12 h (00-12)

%j

Retourne le n° du jour de l'année (001-366)

%m

Retourne le n° du mois (01-12)

%M

Retourne les minutes (00-59)

%S

Retourne les secondes (00-59)

%U

Retourne le n° de semaine dans l'année.( 00 – 53 )

%w

Retourne le n° du jour dans la semaine (dimanche=0)

%y

Retourne l'année sur 2 chiffres (00-99)

%Y

Retourne l'année sur 4 chiffre (ex: 1993)

 

Remarque

   Le nom du fuseau horaire correspond au nom de la clé du fuseau horaire dans la base de registre.

 

   Exemple :

 

Nom du fuseau Horaire

Description

 

Pacific Standard Time

(UTC-08:00) Pacifique (É.-U. et Canada); Tijuana

GMT Standard Time

(UTC) Heure de Greenwich : Dublin, Édimbourg, Lisbonne, Londres

Romance Standard Time

(UTC+01:00) Bruxelles, Copenhague, Madrid, Paris

Russian Standard Time

(UTC+03:00) Moscou, St. Petersbourg, Volgograd

China Standard Time

(UTC+08:00) Pékin, Chongqing, Hong Kong, Urumqi

 

Exemple

WITH  stdlib,  stdstring;

Procedure Exemple;

VAR

   dateLocale : Integer;

   date : GrandeChaine;

   timeZone : GrandeChaine;

   gestHeureEte : Integer;

Begin

   timeZone := "Romance Standard Time";

   gestHeureEte := 1;

   dateLocale := DeformaterDateUTCEnTZ("%d/%m/%Y %H:%M:%S", "01/03/2015 10:00:00", timeZone, gestHeureEte);

   EcritMessageErreur("--- dateLocale:" + Str(dateLocale));

   date := FormaterDateTZEnUTC("%d/%m/%Y %H:%M:%S", dateLocale , timeZone, gestHeureEte);

   EcritMessageErreur("--- dateFormatUTC:" + date);

End Exemple;
