CommandeShellTempo, Fonction

Exécute une commande shell.

L'exécution s'arrête si la durée d'attente est dépassée.

Syntaxe

CommandeShellTempo (Commande : TresGrandeChaine, Duree : Integer ) return T_AttenteTempo

Elément

Description

 

Commande

Commande Système

Duree

Durée d'attente maximum avant l'arrêt de l'exécution de la commande shell

 

Valeur de retour :

0 : La commande shell s'est bien exécutée

1 : Erreur dans la création et le lancement de la commande shell

2 : Le délai d'attente est dépassé

3 : Erreur dans l'attente du résultat

4 : Echec de l'exécution de la commande shell

5 : Le scénario a été supprimé

Exemple

WITH stdlib, stdstring ;

Procedure Test ;

Var

l_tempo:T_AttenteTempo;

l_cmdShell:TresGrandeChaine;

l_res:integer;

 

Begin

l_cmdShell:="COPY /Y  C:\Essai.txt C:\Temp\Essai1.txt" ;   

l_tempo:= CommandeShellTempo(l_cmdShell,10);

l_res:=LitValAttenteTempo(l_tempo);

EcritMessageErreur("Valeur de retour de la commande shell :"+str(l_res));

 

End  Test ;
