NbMaxValeurMemoire, Fonction

Renvoie le nombre maximum de valeurs que la mémoire peut avoir ou -1 si la mémoire n'existe pas.

Syntaxe

NbMaxValeurMemoire(NomMemoire : GrandeChaine) return Integer;

 

Elément

Description

 

NomMemoire

Nom de la mémoire

 

Exemple

WITH stdlib;

PROCEDURE  Exemple;

VAR

NbMax : Integer;

BEGIN

NbMax :=  NbMaxValeurMemoire( "memoire1" ) ;

END Exemple;
