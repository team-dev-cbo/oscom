Racine, Fonction

 

Calcule la racine carrée d'un réel. Renvoie (-1) si le calcul de la racine carrée n'a pu se passer correctement (par exemple, Reel est négatif).

 

 

Syntaxe

 

Racine ( Reel : REAL ) return REAL

 

Elément

Description

 

Reel

Nombre à traiter

 

 

Exemple

 

WITH stdlib ;

PROCEDURE Exemple;

  VAR   Nombre : REAL;

              Resultat : REAL;

 

 BEGIN

     Resultat:=Racine(Nombre);

     IF (Resultat=-1.0) THEN

         & "Problème lors du calcul de la racine." ;

     END IF;

END Exemple;
