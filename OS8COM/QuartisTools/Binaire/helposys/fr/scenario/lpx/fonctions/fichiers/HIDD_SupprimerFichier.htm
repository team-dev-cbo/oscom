SupprimerFichier, Fonction

Suppression du fichier  (avec son chemin).

Syntaxe

SupprimerFichier (CheminFichier : GrandeChaine;  DelaiMax : Integer) return T_Fichier

 

Elément

Description

 

CheminFichier

Nom du fichier (avec son chemin) qui va être supprimé.

DelaiMax

Délai maximum du traitement (en secondes).

Si DelaiMax <= 0, par défaut DelaiMax = 60 (secondes)

 

Valeur de retour

 0 : si le fichier a été supprimé

 1 : si erreur

  ( Le fichier n'existe pas,

    Le chemin d'accès spécifié est introuvable,

    Le chemin réseau n'a pas été trouvé,

    Nom du réseau introuvable,

Accès refusé ... )

 2 : si erreur de timeout

  Le délai maximum (en secondes) a été dépassé avant la fin du traitement.

 4 : si le scénario a été supprimé

 

 

Remarque

Si DelaiMax  <= 0 ,  on impose une valeur par défaut qui est de 60 secondes.

Il faut appeler la fonction ReponseFichier pour récupérer le résultat de la suppression du fichier.

 

 

Exemple

 

With stdlib, stdstring ;

Procedure ExempleFic ;

Var 

l_res : Integer ;

l_reponseFic : T_Fichier;

Begin

// Suppression du fichier

l_reponseFic := SupprimerFichier( "c:\tmp\essai.txt , 10 );

l_res := ReponseFichier(l_reponseFic,1);

 

EcritMessageErreur ("Resultat="+Str(res));

 

// Suppression le fichier qui se trouve sur un réseau distant

l_reponseFic := SupprimerFichier( "\\NomServeur\Repertoire1\Repertoire_2\Fichier1.doc", 10 );

l_res := ReponseFichier(l_reponseFic,1);

 

EcritMessageErreur ("Resultat="+Str(res));

End ExempleFic ;
