XmlFilsAjoute, Fonction

 

Ajoute un fils à l’élément courant de la structure xml.

 

 

Syntaxe

 

XmlFilsAjoute( p_xml : T_Xml ; p_fils , p_valeur : GrandeChaine ) RETURN Integer

 

Elément

Description

 

p_xml

identifiant structure xml

p_fils

Nom du fils

p_valeur

Valeur du fils

 

 

Valeur retournée

0 : OK ,

1 : erreur ajout,

2 : erreur identifiant structure xml.
