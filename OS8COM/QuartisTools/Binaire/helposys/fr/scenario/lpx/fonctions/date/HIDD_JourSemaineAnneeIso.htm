JourSemaineAnneeIso, Fonction

 

Renseigne pour une date donnée : le jour, la semaine et l’année de la semaine, selon la norme ISO.

 

 

Syntaxe

 

JourSemaineAnneeIso( p_date, p_premierJour : Integer ,

@p_jour , @p_semaine , @p_anneeSemaine : Integer ) RETURN Integer

 

Elément

Description

 

p_date

Date à renseigner.

p_premierJour

1er jour de la semaine ( Dim :0, Lun :1, Mar :2, Mer :3, Jeu :4, Ven :5, Sam :6 )

p_jour

( 0 – 6 ) ( Dim :0, Lun :1, Mar :2, Mer :3, Jeu :4, Ven :5, Sam :6 )

p_semaine

( 1 – 53 )

p_anneeSemaine

( 1970 – 2038 )

 

 

Valeur retournée

0 si OK , <0 sinon.

 

Exemple

 

With stdlib;

 

Procedure test3;

 

Var

l_res : Integer;

l_jour , l_semaine , l_annee : Integer ;

l_chaine : GrandeChaine ;

 

Begin

 

 

l_res := JourSemaineAnneeIso( DeformaterTemps( "%D" , "01/01/2000" ) ,0 ,@ l_jour,@ l_semaine ,@ l_annee ) ;

& "01/01/2000" + " Jour:" + Str(l_jour) + " Semaine:" + Str(l_semaine) + " Année:" + Str(l_annee) ;

 

 

End test3;
