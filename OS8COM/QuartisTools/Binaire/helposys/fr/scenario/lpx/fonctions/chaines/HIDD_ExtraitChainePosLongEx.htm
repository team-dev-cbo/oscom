ExtraitChainePosLongEx, Fonction

 

Extrait une partie d'une chaine de grande Taille.

 

 

Syntaxe

 

ExtraitChainePosLongEx( p_chaine : ChaineEx ; p_position, p_longueur : Integer ) RETURN ChaineEx

 

Elément

Description

 

p_chaine

Chaine de caractères

p_position

Position du 1er caractère à extraire dans la chaine ( 1: début de la chaine )

p_longueur

Nombre de caractères à extraire

Exemple

 

WITH stdstring ;

 

PROCEDURE Exemple ;

 

VAR

 l_resultat : Integer ;

 l_nbCar  : Integer ;

 l_chaineEx : ChaineEx ;

 l_sousChaineEx : ChaineEx ;

 

BEGIN

 // copie d'un fichier dans une chaine

 l_resultat := FichierVersChaineEx( "c:\tmp\ca.txt" ,@ l_chaineEx ) ;

 

 IF l_resultat=0 THEN

 

  // longueur de la chaine

  l_nbCar := LongueurEx( l_chaineEx );

 

  // extraction des 5 derniers caractères

  IF l_nbCar>5 THEN

 

  l_sousChaineEx := ExtraitChainePosLongEx( l_chaineEx , l_nbCar-4 ,5 );

 

  END IF ;

 

 END IF ; 

  

END Exemple ;
