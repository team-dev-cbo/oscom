DeplacerFichier, Fonction

Déplacer le fichier source (avec son chemin) vers un fichier destinataire (avec son chemin).

Le nom du fichier destinataire peut être identique ou différent du nom du fichier source.

Si le fichier destinataire existe déjà, il est écrasé par le fichier source.

Syntaxe

DeplacerFichier (CheminFichierSource : GrandeChaine; CheminFichierDestination : GrandeChaine; DelaiMax : Integer) return T_Fichier

 

Elément

Description

 

CheminFichierSource

Nom du fichier Source avec le chemin .

CheminFichierDestination

Nom du fichier Destinataire avec le chemin.

DelaiMax

Délai maximum du traitement (en secondes).

Si DelaiMax <= 0, par défaut DelaiMax = 60 (secondes)

 

Valeur de retour

 0 : si OK

 1 : si erreur

  ( Le fichier source n'existe pas,

    Le chemin d'accès spécifié est introuvable,

    Le chemin réseau n'a pas été trouvé,

    Nom du réseau introuvable ... )

 2 : si erreur de timeout

  Le délai maximum (en secondes) a été dépassé avant la fin du traitement.

 4 : si le scénario a été supprimé

 

Remarque

Si DelaiMax  <= 0 ,  on impose une valeur par défaut qui est de 60 secondes.

Il faut appeler la fonction ReponseFichier pour récupérer le résultat du déplacement du fichier.

 

 

Exemple

 

With stdlib, stdstring ;

Procedure ExempleFic ;

Var 

l_res : Integer ;

l_reponseFic : T_Fichier;

Begin

// Déplacer le fichier source

l_reponseFic := DeplacerFichier( "c:\tmp\essai.txt", "d:\tmp\essai.txt", 10 );

l_res := ReponseFichier(l_reponseFic,1);

 

EcritMessageErreur ("Resultat="+Str(res));

 

// Déplacer le fichier source

l_reponseFic  := DeplacerFichier( "c:\tmp\essai.doc", "d:\tmp\tempo.doc", 5 );

l_res := ReponseFichier(l_reponseFic,1);

 

EcritMessageErreur ("Resultat="+Str(res));

 

// Déplacer le fichier source qui se trouve sur un réseau distant

l_reponseFic  := DeplacerFichier( "\\NomServeur\Repertoire1\Repertoire_2\Fichier1.doc", "d:\tmp\Fichier1.doc", 10 );

l_res := ReponseFichier(l_reponseFic,1);

 

EcritMessageErreur ("Resultat="+Str(res));

End ExempleFic ;
