Message, Fonction

 

Lecture d’un message dans la langue courante.

 

 

Syntaxe

 

Message( p_cle : GrandeChaine ; @ p_message : GrandeChaine ) RETURN Integer

 

Elément

Description

 

p_cle

Clef message

p_message

Chaine destination du message

 

Valeur retournée

0 : OK ,

-1 : Clef inconnue.

-2 : Non défini pour la langue courante.

 

Remarque

Si la mémoire système Code scénario est non vide, la fonction recherche d’abord le message avec la clé définie par le contenu de la mémoire Code scénario-p_cle.

( Ex : si la valeur de Code scénario est « Arrivée » et la valeur de p_cle est « Mess1 » , la clef du message recherché sera « Arrivée-Mess1 ».)

 

Exemple

 

With Mess ;

 

Procedure test1 ;

 

Var

 

 l_langue : GrandeChaine ;

 l_message : GrandeChaine ;

 l_res : Integer ;

 

Begin

 

 

 // message dans la langue courante

 l_res := Message( "M1" , @ l_message );

 

 // message dans la langue anglaise

 l_res := MessageLangue( "M1" , "EN" , @ l_message );

 

 // message dans la langue espagnole

 l_res := MessageLangue( "M1" , "SP" , @ l_message );

 

 

End test1 ;
