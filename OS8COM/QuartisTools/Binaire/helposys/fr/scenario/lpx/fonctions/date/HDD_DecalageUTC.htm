DecalageUTC, Fonction

Renvoie le décalage horaire (en seconde(s)) entre l'UTC(0) et le fuseau horaire passé en paramètre.

L' UTC (0) est un fuseau horaire, sans décalage horaire par rapport à l'UTC (Temps universel coordonné).

Si le fuseau horaire n'existe :

-  un message d'erreur apparaît dans l'interface

- la fonction renvoie 0

Exemple :

Fuseau Horaire  : "Romance Standard Time" ((UTC+01:00) Bruxelles, Copenhague, Madrid, Paris)

Gestion de l'heure d'été :

Heure d'hiver :  Décalage =  + 3600

Heure d'été    :  Décalage =  + 7200

Pas de gestion de l'heure d'été :

Décalage = + 3600

Syntaxe

DecalageUTC ( FuseauHoraire : GrandeChaine ; GestionHeureEte : INTEGER ) return Integer

 

Elément

Description

 

FuseauHoraire

Nom du fuseau horaire

GestionHeureEte

Gestion ou non de l'heure d'été (0 : non, 1 : oui)

 

Remarque

   Le nom du fuseau horaire correspond au nom de la clé du fuseau horaire dans la base de registre.

 

   Exemple :

 

Nom du fuseau Horaire

Description

 

Pacific Standard Time

(UTC-08:00) Pacifique (É.-U. et Canada); Tijuana

GMT Standard Time

(UTC) Heure de Greenwich : Dublin, Édimbourg, Lisbonne, Londres

Romance Standard Time

(UTC+01:00) Bruxelles, Copenhague, Madrid, Paris

Russian Standard Time

(UTC+03:00) Moscou, St. Petersbourg, Volgograd

China Standard Time

(UTC+08:00) Pékin, Chongqing, Hong Kong, Urumqi

 

Exemple

WITH stdlib;

Procedure Exemple;

VAR

    NomeTZI:GrandeChaine;

    Offset : Integer;

Begin

   // Fuseau Horaire : Paris Madrid (UTC+1)

   NomTZI := "Romance Standard Time";   

   Offset := DecalageUTC ( NomTZI, 1 );

   EcritMessageErreur(NomTZI +" - Offset = "+Str(Offset));

End Exemple;
