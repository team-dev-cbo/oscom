Création d'un scénario graphique

 

Un scénario graphique est constitué d'actions.

Cliquer et déposer une action dans le scénario graphique et définir les propriétés de l'action.

 

Pour se déplacer dans la liste des actions du scénario graphique :

Sélectionnez une action (en cliquant dessus)

Utiliser les icônes :

précédent,  suivant, père,  fils
