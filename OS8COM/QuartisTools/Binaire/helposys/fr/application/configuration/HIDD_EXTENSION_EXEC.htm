Type Extension-Exécutable

 

Principe :

Pour le serveur Web, s'il autorise les CGI, associer un exécutable à une extension.

 

Propriétés :

Extension

Nom de l'extension

Programme

Exécutable associé à l'extension

 

Exemple :

 Extension = "php"

 Programme =  "c:\sfao\php4\php.exe"
