Imprimer  une documentation

 

Principe :

Création dynamique de la documentation, dans la langue du demandeur, de toute l'application, d'un module particulier, d'un groupe particulier ou d'un élément.

 

Propriétés :

Type d'impression

Résumé : La documentation est partielle.

Détaillé  : Impression de toute la documentation.

Destination

Envoie la documentation à une imprimante ou dans un fichier.
