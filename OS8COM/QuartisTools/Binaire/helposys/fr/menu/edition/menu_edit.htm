Listes des commandes du menu Edition

 

Le menu Edition propose les commandes suivantes :

 

Couper

Supprime les données sélectionnées du document et les place dans le Presse-papiers.

Copier

Copie les données sélectionnées du document vers le Presse-papiers.

Coller

Colle les données contenues dans le Presse-papiers dans le document.

Rechercher

Recherche le mot dans le scénario LPX.

Remplacer

Remplace un mot par un autre pour un scénario LPX.

Supprimer

Supprime un objet graphique sélectionné pour une fenêtre.
