Commande Copier

Menu "Edition/Copier"

 

Utilisez cette commande pour copier les données sélectionnées dans le Presse-papiers. Cette commande n'est pas disponible si aucune donnée n’a été préalablement sélectionnée.

 

Lorsque vous utilisez cette commande, le contenu du Presse-papiers se trouve remplacé par les nouvelles données copiées.

 

Pour l'élément fenêtre, on copie un objet graphique.

 

Raccourcis ( sauf pour l'élément fenêtre )

Barre d'outils : 

Touches :  CTRL+C
