Utilisation du navigateur

 

Envoi d'un message vers le serveur web :

2 techniques sont possibles :

par l'adresse URL :

2 syntaxes :

 http://NomMachineServeur/osys?osys=Message

ou

 http://NomMachineServeur/osysMessage?

EXEMPLE :

http://oceane/osys?osys=bonjour

ou

http://oceane/osysbonjour?

 

par un formulaire :

Le formulaire doit contenir les caractéristiques suivantes :

Méthode du formulaire : POST

Action du formulaire : OSYS

Nom du champ comportant le message : OSYS.

EXEMPLE : code Html d'une zone de saisie dans un formulaire

<form action="OSYS" method="POST">

<p><input type="text" size="20" name="OSYS"></p>

</form>

 

Remarque: Le périphérique du Message envoyé est le périphérique Clavier

 

Chargement d'une page Html :

Cette page a été définie par un éditeur spécifique

Le code source de cette page est sauvegardé dans un fichier ( par ex: exemple.htm )

Ce fichier se situe dans un sous-répertoire ( par ex: Essai ) du répertoire de base du serveur Web

( voir Configuration du serveur Web )

 

Pour charger cette page dans le navigateur, la syntaxe de l'adresse http est la suivante:

http://NomMachineServeur/Essai/exemple.htm

 

Chargement dynamique de page Html :

Le chargement de pages dynamiques s'effectue par le biais de programmes LPX.

Voir les fonctions Html dans la Liste des fonctions LPX

 

Lancement d'un programme externe :

L'option Autorisation CGI doit être cochée. ( voir Configuration du serveur Web )

On suppose que l'exécutable (Exec.exe) se trouve dans le répertoire de base du serveur Web.

http://NomMachineServeur/Exec.exe

 

Chargement de la page par défaut :

Une page par défaut nommée default.htm est chargée si l'adresse du navigateur est la suivante :

http://NomMachineServeur

 

Le fichier (default.htm ) doit se trouver à la racine du répertoire de base du serveur Web
