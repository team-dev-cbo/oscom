Affichage de la trace

 

Affichage d'un message de suivi du fonctionnement de l'application provenant du serveur sous la forme :

"Date - Module - Utilisateur - Message"

avec  :

Date : Format de la date sous la forme "HH:MM:SS:Millisecondes"

Module : Le message provient d'un module particulier

Utilisateur : Nom de l'utilisateur concerné

Message :  Message de suivi

 

 

Signification de la trace suivant la couleur :

 

Rouge

Message d'erreur

Orange

Message de warning

Vert

Message d'information

Bleu
