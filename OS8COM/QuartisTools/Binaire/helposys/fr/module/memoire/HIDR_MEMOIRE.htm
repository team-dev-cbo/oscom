Mémoire

 

Une mémoire est définie par :

 

Nom de la mémoire :

Ce nom peut être composé de lettres, de chiffres ainsi que des espaces mais pas d'accent.

 

Description

Un commentaire est accessible en fonction de langue désirée.

 

Mode :

Mode Simple : La valeur accédée est toujours la première

Mode FIFO :  la valeur accédée est toujours la première, l’écriture d’une valeur ajoute cette valeur avec l’indice 1 (première position) en décalant toutes les valeurs (une valeur peut être perdue si le nombre max de valeur est atteint.

Un troisième mode permet de définir la mémoire comme un mémoire dynamique : elle contient des valeurs dite système  (date, heure, nom machine...) ou une valeur calculé lors de chaque lecture (gestion identique pour les deux cas mais modification de la formule de calcul impossible pour les mémoires système). Le contenu de ces mémoires ne peut pas être modifié par un utilisateur et est calculé à chaque accès.

 

Visibilité :

Globale : La mémoire est globale à l’application.

Groupe : La mémoire est propre à un groupe d'utilisateurs.

Utilisateur : La mémoire est la propriété d’un utilisateur par défaut (une valeur par utilisateur).

 

 Nombre maximum de valeurs dans une mémoire

 Une mémoire peut contenir une ou plusieurs valeur. Lorsqu’une mémoire contient plusieurs valeurs, chaque valeur peut être accédée par son indice (de 1 à n).  Par contre, si l’indice n’est pas précisé, deux modes de fonctionnement sont possibles : le mode simple ou le mode fifo

 

 Valeur par défaut

 

 Gestion ou non des valeurs dans un fichier

Le contenu de la mémoire est contenu en mémoire par défaut, mais peut être géré dans un fichier (cas d’un nombre de valeurs important).

 

 

 

Remarque :

 

La lecture d’une mémoire n’ayant jamais été définie retourne sa valeur par défaut ( valeur modifiée si nécessaire pour que le type de donnée corresponde).

Une catégorie de mémoires sont différentes des autres : Les mémoires systèmes (du groupe des mémoires systèmes).

   Elles sont créées par le moteur et ne peuvent être ni modifiées ni supprimées par l'utilisateur.
